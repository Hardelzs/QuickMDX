## What is QuickMDX?

QuickMDX is a lightweight tool that empowers developers to create stunning documentation and project using MDX (Markdown with JSX). With the fusion of Markdown's simplicity and JSX's interactivity, you can craft informative and engaging documentation that shines.

## Installation

Let's get you started in no time! First, ensure you have Node.js and npm installed. Then, install QuickMDX globally:

```bash
npm install -g quick-mdx
```

## Usage

Once you have QuickMDX installed, creating a new documentation project is a breeze:

```bash
quick-mdx create my-docs
```

This command sets up a new project, already configured to harness the power of MDX.

### Embedding Components

Add interactive elements with JSX components:

```jsx
import Button from './components/Button';

<Button>Click me</Button>
```

### Styling

Apply custom styles with CSS classes directly to components:

```jsx
<Button className="primary-button">Primary</Button>
```
